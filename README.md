# QRCodeTool
## 简述
利用Python编写的图形化二维码解析+生成工具  
只是一个Python练手项目，代码结构并不是非常清晰。
## 功能
* 区域截图并识别二维码
* 识别图片文件中的二维码
* 文本信息转换为二维码
* 兼有保存截图的功能
